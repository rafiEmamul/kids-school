Question-3: What is event delegation in the DOM?
Event delegation in the DOM is a technique where attach a single event listener to a parent element, rather than attaching individual event listeners to each of its child elements. This allows to handle events for multiple child elements efficiently.
Question-4: What is event bubbling in the DOM?
Event bubbling is a fundamental concept in web development that describes the order in which events are handled in the Document Object Model (DOM) when nested elements are involved.
